## Installation or Getting Started

Run the following command in the terminal:

	git clone https://github.com/Nishkarsh01/AttendanceWarden..git
or download the zip file from github.
    

## Usage
After extracting the files,

    cd AttendanceWarden.
    npm install 
    node App.js
